Mock Exam — Page 1

This is a test page for the Exam Proctor extension. Proctoring should auto-start here.

1. Fullscreen gate

When this page loaded out of fullscreen, the extension should have shown a blocking “Fullscreen required” overlay. Click its button to enter fullscreen and begin.

You can also press Esc to exit fullscreen. Each exit is one strike.

2. Clipboard & shortcut blocking

Try to copy this text, right-click, or press Ctrl+P / Ctrl+S / Ctrl+U / F12. Each should be blocked and logged.

3. Tab / window / navigation

Each of these should be recorded as a violation (counts toward the 3-strike limit):

  • Open Page 2 in a new tab → “New tab” + “Tab switch”.
  • Click another application / minimise → “Window blur”.
  • Navigate to Page 2 in this tab → leaves the exam URL → session ends.
Open Page 2 (new tab) Go to Page 2 (this tab — ends exam)

4. Strike escalation

Trigger violations and watch the on-page warning escalate:

  • 1st violation → red banner that auto-hides after ~5 seconds.
  • 2nd violation → darker banner that stays until you click it.
  • 3rd violation → blocking “Exam ended” overlay + the session stops.

A JSON log is auto-saved to Downloads/exam-proctor-reports/ and rewritten on every event.
